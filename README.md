## Project Overview

This project presents an interactive Tableau dashboard built using an e-commerce sales dataset. The dashboard provides insights into sales performance, customer behaviour, product categories, and geographical trends.

## Dashboard Features

- KPI cards for Revenue, Profit, Orders and Customers
- Monthly Sales Trend
- Revenue by Product Category
- Customer Segment Performance
- Sales by Country
- Interactive filtering

## Tools Used

- Tableau Public
- CSV Dataset

## Key Insights

- Electronics generated the highest revenue.
- Sales remained relatively consistent throughout the year.
- Returning customers contributed significantly to total revenue.
- Revenue varied across different countries.
